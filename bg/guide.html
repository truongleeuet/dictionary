在线词典助手-使用说明

欢迎使用本插件，以下是插件简要说明，如需完整文档或插件最新信息，请访问 项目主页

  1. 打开任意网页，将鼠标移动到想要查询的单词上。
  2. 点击拖动或者双击鼠标选取单词或者短语。如果单词文本是一个链接，可按下取词热键自动取词。
  3. 选词后，插件会弹出释义窗口显示在单词旁边。
  4. 左击图标，可以快速启用/禁用插件，选择取词热键，或者选择当前词典。
  5. 右击图标，然后选择"选项"可打开选项页，以进一步定制化插件。

本插件主要以脚本获取在线词典信息，下面是部分脚本支持的词典名称，完整内容详见脚本清单

  1. 简明词典：有道简明英汉，百词斩图文词典
  2. 权威词典：柯林斯双解，牛津双解，剑桥双解
  3. 其他词典：法语助手、西语助手、千亿俄汉、汉典汉语
  4. 其他双语：柯林斯英法/法英、剑桥英法/法英词典

根据广大用户的请求，增加了Ankiweb制卡接口。但使用Ankiweb接口制卡，可能因违反TOS而被封禁帐号，请谨慎使用，责任自负

注意：ankiconnect官版不支持同一单词重复制卡。想重复制卡，详见这篇博客文章

Online Dictionary Helper Guide

Thanks for installing. Here is the brief introduction. If you need more details, please visit Project Home

  1. Open any given web page, move mouse cursor to the word.
  2. Click-Drag/Double Click mouse button to select word and phrase, or press Hotkey to automatically select word in case it's a link.
  3. A popup window will show up displaying the word definition.
  4. Left click icon, you may quickly Enabled/Disable this extension, choose Hotkey for automatical text selection, or select current dictionary.
  5. Right click icon, select "Options", you may open options page for further setting

The extension mainly uses script to retrive data from online dictioanry. Below is part of scripts list. For detail, please check Full List

  1. Concise：encn_Youdao, encn_Baicizhan
  2. Learner：encn_Collins, encn_Oxford, encn_Cambridge
  3. Others：frcn_Eudict, escn_Eudict, rucn_Qianyi, cncn_Zdic
  4. Bilingual: Collins(enfr/fren), Cambridge(enfr/fren)

Added Ankiweb interface based on massive request. But Ankiweb account may be disabled according to Ankiweb TOS, please use this function at your own risk

Note: Sometime, you occasionally can not create cards(not always), actually it’s not extension’s problem. Because it does not allow duplicated cards(with same sorted field) creation in ankiconnect by default. for more detail, please check this Blog Article
